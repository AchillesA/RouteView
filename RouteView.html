Mobile Version RouteView - Google StreetView Player - Virtual Ride using Google Street View  Github Project
This will launch the mobile version (Android, IOS).

Please note that the mobile version is a simplified and stripped down version,
which only allows to play a route between two points (no waypoints).
This software is 100% open-source and hosted on Github

Click here if you want to see the source.


Please note that this software is still in Alpha.
Some features are still missing, or may not yet be fully functional,
or they could simply be broken by the latest changes I made...

I am working on it on my spare time, so please bear with me!

Once a route has been defined, this option allows you to create a GPX file (GPS exchange format).
This file can then be uploaded to a GPS device (for instance a Garmin).

This feature is highly untested, and provided at your own risk!

Only tested with my Garmin Nuvi 2689LMT 6" that I use on my RoadKing.
Step : 150 meters - 575 feet
Interval : 750 milliseconds
This slider lets you define the step in the route.

Default is 150 meters, meaning that each Street View image are 250 meters apart.
This slider lets you define the time interval between each Street View image.

Default is 750 milliseconds, meaning that each Street View image is displayed for 750 milliseconds.
This allow you to change Steps and Intervall between each image.
